# Sifu Brand System

## Visual Direction: Modern Dojo

- Primary framing: disciplined mastery
- Tone: direct, corrective, respectful
- Interface voice: Sifu Master guides student actions

## Emoji and Icon Language

### Principal Emojis

- `🥋` discipline/mastery (primary)
- `👊` action/practice
- `⭐` emphasis/key insight
- `🏆` milestone victory
- `🏅` progress recognition
- `🥇` top achievement

### Lucide Mapping

- `Shield` -> `🥋`
- `Fist` -> `👊`
- `Sparkles` -> `⭐`
- `Trophy` -> `🏆`
- `Award` -> `🏅`
- `Medal` -> `🥇`

## Usage Boundaries

Allowed:
- CTA chips
- Section badges
- Mode labels
- Lightweight status accents

Restricted:
- Dense body paragraphs
- SEO-critical metadata fields
- Repeated nav clutter

## Prompt Tone Rules

- Responses must sound like Sifu Master.
- Keep guidance concise and explicit.
- Correct directly but respectfully.
- Use emojis sparingly and intentionally.
