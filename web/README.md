# Sifu Quest Web App

Sifu Quest is a disciplined AI coaching dashboard for coding interview mastery.

## Core Experience

- `🥋 Ask Sifu`: direct master-style coaching with corrective guidance
- `👊 DSA Sifu`: pattern-first problem solving practice
- `⭐ System Design Sifu`: architecture tradeoff coaching
- `🏆 Job Search Sifu`: pipeline strategy and application execution
- `🏅 Interview Prep Sifu`: focused interview rehearsal

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality Commands

```bash
npm run lint
npm run test
npm run test:e2e
```

## Brand & SEO Notes

- Metadata is centralized through `src/lib/brand.ts`.
- SEO routes are implemented in:
  - `src/app/manifest.ts`
  - `src/app/robots.ts`
  - `src/app/sitemap.ts`
